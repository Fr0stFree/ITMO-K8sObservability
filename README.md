# Kubernetes Observability

Comparative analysis of Kubernetes observability stacks: Prometheus Stack, ELK/OpenSearch, VictoriaMetrics/Mimir/Thanos, Tempo/Jaeger, Vector/Alloy, Datadog, New Relic.
Includes IaC, demo app, dashboards, load testing, performance benchmarks, TCO analysis and recommendations.
